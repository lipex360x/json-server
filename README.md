Start JSON-Server: `json-server -w filename.json`

* Optional settings ()

-H: `ip [default: localhost]`
-d: `delay in ms`
-p: `port`

See more with `-h`

 ---

* JSON File Example
```json
{
  "users": []
}
```

* HTTP GET Request

> http://localhost:3000/users
